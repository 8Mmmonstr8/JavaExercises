Пишем игру «Змейка» на Java

Введение

Java – объектно-ориентированный язык программирования высокого уровня, разработанный компанией Sun Microsystems. Впервые был официально представлен в 1995 году. В настоящий момент — один из самых популярных языков программирования. Позволяет вести разработку в четырех направления: настольные системы (desktop), интернет (web), мобильные системы (Android), встроенные системы (embedded systems).

Исходный код на Java компилируется в байт-код, выполняемый виртуальной машиной Java (JVM), которая существует для всех платформ: Windows, Linux, Mac.

Инструменты

Для разработки на Java необходимо скачать и установить как минимум один инструмент — JDK (Java Development Kit).
